# 隐私政策 (Privacy Policy)

**最近更新日期：** [2026/1/9]

本隐私政策旨在说明 **ChatGPT Tools**（以下简称“我们”或“本插件”）如何收集、使用和披露您的信息，以及您对这些信息拥有的选择权。

## 1. 我们收集的信息

### A. 用户生成内容（本地处理）
本插件允许您导出和备份 ChatGPT 的对话记录。此数据直接从您访问的网页 (chatgpt.com) 读取，并完全在您的浏览器**本地**处理，以生成可供下载的文件（JSON、Markdown、HTML）。**我们不会收集、存储或传输您的聊天记录到任何我们自己的服务器。**

### B. AI 预测功能的输入数据
本插件包含“AI 自动补全/预测”功能。当且仅当您在插件设置中开启此功能时，您在输入框中键入的文本以及当前的对话上下文（最近几条消息）会被发送给第三方 AI 服务商以获取建议。

### C. 插件设置
我们会将您的偏好设置（例如：自动备份间隔、是否开启液态主题、是否开启防误触提醒）保存在您的浏览器本地存储中 (`chrome.storage.local`)。

## 2. 我们如何使用您的信息

*   **导出与备份：** 用于格式化您的数据并保存到您的本地设备。
*   **功能配置：** 读取本地存储的设置以应用您的个性化偏好。
*   **AI 功能服务：** 将必要的文本片段发送给 AI 提供商，仅用于生成下一句预测或文本补全。

## 3. 第三方服务

为了提供智能补全功能，我们使用了以下第三方服务：

*   **DeepSeek API:**
    *   当“AI 自动补全”或“预测”功能处于**开启状态**时，您的输入文本和上下文数据将通过加密连接发送至 DeepSeek API (`https://api.deepseek.com`) 进行处理。
    *   这些数据仅用于生成即时的回复建议，不会被本插件用于任何其他用途。
    *   关于 DeepSeek 如何处理数据，请参阅 DeepSeek 的隐私政策。

**您的控制权：** 如果您不希望将任何数据发送给 DeepSeek，您可以在插件的弹出菜单（Popup）中随时关闭“AI 自动补全”开关。关闭后，插件将完全停止与 DeepSeek API 的通信。

## 4. 数据安全

*   **本地优先：** 本插件的绝大多数核心功能（导出、UI 主题美化、自动备份）均完全在您的本地浏览器环境中运行。

## 5. 政策变更

我们需要时可能会更新本隐私政策。如有更改，我们将在此页面发布新的隐私政策以通知您。
